# Task Persistence Module

Task persistence and continuity tracking. Maintains session state, progress tracking, and task history across multiple AI interactions.

| File | Description |
|------|-------------|
| `active-tasks.md` | Currently active task tracking |
| `continuity-log.md` | Cross-session continuity log |
| `finding-dossier.md` | Finding dossier structure |
| `progress-tracker.md` | Task progress tracking |
| `recon-register.md` | Reconnaissance findings register |
| `retrospective.md` | Session retrospective notes |
| `session-protocol.md` | Session start/end protocol |
| `session-states.md` | Session state machine definitions |
| `task-history.md` | Historical task records |
