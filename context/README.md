# Context Module

Session context files that track active hunting state, chain primitives, and lessons learned during a hunt session.

| File | Description |
|------|-------------|
| `active-target.md` | Current active target details |
| `chain-primitives.md` | Discovered chain primitives |
| `hunt-session.md` | Active hunt session state |
| `lessons-log.md` | Lessons learned during hunts |
| `target-registry.md` | Target registration and metadata |
| `tool-inventory.md` | Available tools inventory |
| `vuln-class-checklists.md` | Vulnerability class checklists |
