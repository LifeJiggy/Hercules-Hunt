# Memory Module

Persistent memory schemas for AI session state. Hydrated during hunt sessions to maintain continuity across multiple AI interactions.

| File | Description |
|------|-------------|
| `discoveries.md` | Track of findings and discoveries across sessions |
| `lessons-log.md` | Lessons learned and methodology refinements |
| `persistence.md` | Persistence framework for cross-session state |
| `session-state.md` | Current session context and progress tracking |
| `target-registry.md` | Registry of targets and their metadata |
| `technical-notes.md` | Technical reference notes and cheat sheets |
| `universal.md` | Universal context shared across all sessions |
