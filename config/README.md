# Configuration Files

Configuration files for hunt sessions and tool behavior.

| File | Description |
|------|-------------|
| `hunter-config.json` | Hunt session configuration |
| `notifications.json` | Notification settings |
| `profiles.json` | Tool and target profiles |
| `targets.json` | Target definitions |
| `tools-config.json` | Tool behavior configuration |
| `waf-bypass.json` | WAF bypass techniques and payloads |
| `wordlists.json` | Wordlist paths and metadata |
