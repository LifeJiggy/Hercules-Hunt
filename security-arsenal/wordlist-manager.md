Organized wordlists and wordlist management for bug bounty hunting.

## WORDLIST MANAGEMENT PHILOSOPHY

```
WORDLIST LIFECYCLE:
1. CONVERGE — Start with minimal wordlist (top 100)
2. EXPAND — Add target-specific keywords
3. FILTER — Remove dead responses, analyze useful paths
4. RECURSE — Use discovered paths to generate more
5. CUSTOMIZE — Build target-specific wordlist
```
