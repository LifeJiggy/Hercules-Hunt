# Scripts Module

Installation and setup scripts for the Hercules-Hunt toolkit across Windows, Linux, and macOS.

| File | Description |
|------|-------------|
| `hunt.sh` | Linux/macOS hunt launcher script |
| `install-community-skills.sh` | Community skills installation script |
| `install.ps1` | Windows PowerShell installation script |
| `install.sh` | Linux/macOS installation script |
| `jiggy-adapter.py` | Python adapter for Jiggy integration |
